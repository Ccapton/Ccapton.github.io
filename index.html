android开发之无线连接功能

发表于 2019-12-22 | 分类于 Android开发 | 阅读次数:

分享缘由

前段时间和同事在一起用了一下AndroidStudio的无线连接功能，然后就去琢磨了一下，发现原来它使用的是adb命令进行操作的，于是我两继续学习了一番

关于adb

adb 中有个开启手机 tcpip协议服务的命令！！！

adb tcpip 5555

即以 5555 端口开放 手机的无线连接功能！！！（需要手机adb调试模式连接上电脑）

然后在你的电脑端打开terminal终端，连接到这台手机上，前提是电脑、手机必须在同一个局域网下。例如你的手机在当前局域网中ip为 192.168.1.102 ，则执行

adb connect 192.168.1.102:5555

这时你的电脑（客户端）已经连接到手机（服务端）了

然后在AndroidStudio中就可以发现它啦，进行调试、运行app非常方便。

使用细节

  • 手机作为服务端只能被一个电脑（客户端）连接，换电脑连接当前手机时需要在上一个电脑执行 adb disconnect 192.168.1.102:5555
  • 手机一旦使用 adb tcpip 5555 命令开启服务后，只能等下次重启或关机才会结束tcpip服务。也就是说，只要手机不重启、关机，就不用再次使用 adb tcpip来开启无线连接服务了

adb更多好玩的用法

adb devices
查看当前已连接Android设备（包含虚拟机）

adb pull /sdcard/Android/test.txt /Users/ccapton/Desktop/test.text 
# 从手机里复制test.txt到电脑桌面

adb push /Users/ccapton/Desktop/test.text /sdcard/Android/test.txt 
# 电脑桌面test.txt复制到手机去

adb install /Users/ccapton/Desktop/test.apk 
# 安装test.apk到你的手机

adb uninstall com.ccapton.test 
# 卸载包名为 com.ccapton.test 的app

adb shell
# 进入手机的终端terminal

# 当有多个Android设备时，以上命令需要指定设备，即带上 -s 设备名，例如
adb -s 192.168.1.113:5555 pull /sdcard/Android/test.txt /Users/ccapton/Desktop/test.text

科学上网最优方案 ( VPS + Lotsever加速 + ShadowSocks / Brook )

前言

关于 VPS (Virtual Private Server 虚拟专用服务器) ，搞技术的朋友都不会陌生，这个服务最近今年在国内外都很火热。我这几年科学上网后，个人感觉：

关于购买vpn服务上网

  • 搭建、使用vps来科学上网的性价比似乎远不及买一个vpn服务账号来的高
  • vpn服务商稳定性不能保证，因为之前买过一些套餐，有时会很慢才能脸上，甚至最后服务商直接跑了。服务供应商如果不能稳定的生存下去，那么你买的服务也随流产
  • 每家vpn服务商提供的客户端都不一样，导致每次换服务商就要下载新的客户端

关于自建vps上网

  • 如果选购的vps带宽不高的话，看youtube视频的速度、清晰度也不如人意
  • 选对一个实惠且稳定的vps供应商很重要，由于不是用于企业服务，仅仅是个人用来科学上网，vps除了高带宽和基本的cpu性能，根本不需要多余的配置
  • 不要选购大品牌vps，因为买的人很多，国外ip流量很大，会被中国政府屏蔽ip! 例如：vultr，虽然vultr的计费方式是按分钟计费，可以重新部署新的系统来实现更换ip的目的，但是我试了很多次，每次分配的ip都无法ping通。于是我放弃了，浪费了精力、时间和金钱

所以选对一个相对冷门又稳定的vps供应商很重要，配合Lotsever加速技术（提高服务器处理网络数据的效率），再搭建ss/brook服务器，客户端就能享受相对稳定、快速的视频观看体验了
